# Проект приложения создания базы данных с вакансиями платформы HeadHunter.ru
## Описание:
**Данное приложение предполагает создание базы данных с вакансиями по заданным ID работодателей.**
## Цель проекта:
Реализовать приложение, который на бэкенде будет создавать и выводить базу данных с информацией о вакансиях.
## Установка:
1. Клонируйте репозиторий:
```python
git@github.com:PetrMaev/course_project_3.git
```
2. Установите зависимости:
```python
poetry install -r requirements.txt
```
## Использование:
Для запуска программы введите в терминале следующую команду:
```python
python main.py
```
## В проекте реализованы:
1. Класс HeadHunterAPI для работы с платформой hh.ru. Класс подключается к API, получает вакансии и записывает в json-файл.
2. Класс JSONFileManager для чтения json-файла.
3. Класс DBCreater для создания и заполнения базы данных данными из json-файла.
4. Класс DBManager для работы с данными базы данных.
5. Функция взаимодействия с пользователем.
